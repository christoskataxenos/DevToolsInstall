# DevTools Installer

A professional Windows desktop application built with Python and Tkinter for managing development environments. It simplifies the installation of various development tools, ensures consistency through "Stacks," and provides robust backup/restore capabilities for developer settings.

[Ελληνικά](#ελληνικά)

## Features

- **Automated Installation**: Leverages `winget` for silent, reliable installation of 100+ developer tools and essential apps.
- **Environment Stacks**: Predefined bundles (Python/AI, React/Web, DevOps, etc.) for one-click environment setup.
- **Settings Backup & Restore**: Easily backup and restore settings for VS Code, Cursor, Windsurf, and other common developer tools.
- **Modern UI/UX**: A responsive, grid-based dark/light interface with custom-designed widgets and smooth scrolling.
- **European Focus**: Includes high-quality European and open-source alternatives (Vivaldi, Proton, pCloud, etc.) with a focus on privacy.

## Why the Greek Language?

The application's interface includes descriptions and labels in **Greek**. This was a deliberate design choice to provide a localized, user-friendly experience for Greek-speaking developers and enthusiasts, making the tool more accessible and intuitive for the local ecosystem while maintaining professional standards.

## Installation

1. Ensure you have Python 3.10+ installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/your-username/DevToolsInstaller.git
   cd DevToolsInstaller
   ```
3. Run the setup script:
   ```bash
   ./setup.bat
   ```
4. Run the application:
   ```bash
   python DevToolsInstaller.py
   ```

# Ελληνικά

Μια επαγγελματική desktop εφαρμογή για Windows, κατασκευασμένη με Python και Tkinter, για τη διαχείριση προγραμματιστικών περιβαλλόντων. Απλοποιεί την εγκατάσταση εργαλείων, εξασφαλίζει τη συνοχή μέσω των "Stacks" και παρέχει δυνατότητες backup/restore για τις ρυθμίσεις των προγραμματιστών.

## Χαρακτηριστικά

- **Αυτοματοποιημένη Εγκατάσταση**: Χρησιμοποιεί το `winget` για την αθόρυβη και αξιόπιστη εγκατάσταση άνω των 100 εργαλείων και εφαρμογών.
- **Environment Stacks**: Προκαθορισμένα πακέτα (Python/AI, React/Web, DevOps, κ.λπ.) για ρύθμιση περιβάλλοντος με ένα κλικ.
- **Backup & Επαναφορά Ρυθμίσεων**: Δημιουργήστε εύκολα αντίγραφα ασφαλείας και επαναφέρετε ρυθμίσεις για VS Code, Cursor, Windsurf και άλλα εργαλεία.
- **Σύγχρονο UI/UX**: Μια responive διεπαφή βασισμένη σε grid, με σκοτεινό/φωτεινό θέμα και custom widgets.
- **Έμφαση στην Ευρώπη**: Περιλαμβάνει ευρωπαϊκές και open-source εναλλακτικές (Vivaldi, Proton, pCloud κ.λπ.) με έμφαση στην ιδιωτικότητα.

## Γιατί Ελληνικά;

Η διεπαφή της εφαρμογής περιλαμβάνει περιγραφές και ετικέτες στα **Ελληνικά**. Αυτή ήταν μια συνειδητή σχεδιαστική επιλογή για την παροχή μιας τοπικής, φιλικής προς τον χρήστη εμπειρίας για τους Έλληνες προγραμματιστές, καθιστώντας το εργαλείο πιο προσιτό και διαισθητικό για το τοπικό οικοσύστημα.
